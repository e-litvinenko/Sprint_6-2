# Автотесты для Яндекс.Самокат https://qa-scooter.praktikum-services.ru/


## Тестовые сценарии

### 1. **TestFAQ** - Проверка раздела "Вопросы о важном"
- **test_faq_questions** - Проверка выпадающего списка вопросов (8 вопросов)
  - При клике на вопрос открывается соответствующий текст
  - Каждый вопрос проверяется отдельно через параметризацию
  - Проверка корректности текста ответов для всех 8 вопросов

### 2. **TestOrderScooter** - Заказ самоката
- **test_order_scooter_positive** - Позитивный сценарий заказа самоката
  - Тестирование через верхнюю кнопку "Заказать" (order_button_position="top")
  - Тестирование через нижнюю кнопку "Заказать" (order_button_position="bottom")
  - Полный сценарий: заполнение формы → подтверждение → успешный заказ
  - Тестирование с двумя наборами данных через параметризацию

### 3. **TestOrderScooter** - Проверка навигации
- **test_scooter_logo_navigation** - Проверка перехода на главную через логотип Самоката
  - Клик на логотип "Самоката" → переход на главную страницу "Самоката"
- **test_yandex_logo_navigation** - Проверка перехода на Дзен через логотип Яндекса
  - Клик на логотип "Яндекса" → открытие Дзен в новом окне через редирект

## Page Object классы

- **BasePage** - базовый класс с общими методами (ожидания, поиск элементов, клики)
- **MainPage** - главная страница с кнопками заказа и логотипами
- **OrderPage** - страница оформления заказа (две страницы формы)
- **FaqPage** - раздел "Вопросы о важном" (аккордеон с вопросами)


## Установка и запуск

# Установить зависимости
pip install -r requirements.txt

# Запустить все тесты
pytest tests/

# Запустить тесты с подробным выводом
pytest tests/ -v

# Запустить конкретный тестовый файл
pytest tests/test_faq.py 


# Для генерации Allure отчета:
pytest tests/ --alluredir=allure-results
allure generate allure-results -o allure-report --clean
allure open allure-report# Sprint_6
